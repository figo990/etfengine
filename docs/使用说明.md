# ETFEngine 使用说明

## 快速开始

### 1. 环境准备

```bash
# 创建虚拟环境
python -m venv .venv

# 激活（Windows）
.venv\Scripts\activate

# 安装依赖
pip install -e ".[dev]"
```

### 2. 初始化数据

```bash
python scripts/init_data.py
```

该命令会从 AkShare 拉取以下数据到本地 DuckDB 数据库：
- `etf_universe.yaml` 中所有 ETF 的历史日线数据（K线）
- 主要宽基指数估值数据（PE、股息率）
- 中美国债收益率历史数据

### 3. 启动 Dashboard

```bash
streamlit run src/dashboard/app.py
```

浏览器自动打开 `http://localhost:8501`。

### 4. 外盘科技龙头季报（可选）

从 **SEC EDGAR** 拉取 `config/overseas_earnings.yaml` 中配置的美股科技龙头 **季报结构化指标**（营收、净利润、EPS、同比等），Dashboard 中打开 **「外盘季报」** 页面查看。

```bash
python scripts/update_overseas_earnings.py
```

- 请在 `config/overseas_earnings.yaml` 中填写合规的 `sec_user_agent`，或设置环境变量 `SEC_EDGAR_USER_AGENT`（SEC 强制要求）。
- **LLM 解读**与智能资讯共用 `DEEPSEEK_API_KEY`；未配置时仍展示规则生成的中文事实摘要。

## 自定义持仓 ETF

### 修改组合配置

编辑 `config/portfolio.yaml`：

```yaml
portfolio:
  name: "我的ETF组合"        # 组合名称
  total_capital: 200000      # 计划总资金(元)

  holdings:
    - etf: "510300"          # ETF代码
      name: "沪深300ETF"     # 显示名称
      target_weight: 0.30    # 目标权重（0~1，总和=1）
    - etf: "513100"
      name: "纳指100ETF"
      target_weight: 0.25
    - etf: "515080"
      name: "中证红利ETF"
      target_weight: 0.25
    - etf: "518880"
      name: "黄金ETF"
      target_weight: 0.20

  rebalance:
    trigger: "drift"           # drift=偏离触发 | periodic=定期 | both
    drift_threshold: 0.05      # 偏离超过5%触发再平衡
    periodic_frequency: "monthly"
    min_trade_amount: 1000     # 最小交易金额

  risk_limits:
    max_drawdown_alert: 0.15   # 最大回撤预警线
    max_single_position: 0.40  # 单只ETF最大仓位
    min_single_position: 0.05  # 单只ETF最小仓位
```

### 添加新 ETF 到标的池

如果你要追踪的 ETF 不在默认标的池中，编辑 `config/etf_universe.yaml`：

```yaml
etf_universe:
  broad_market:
    - code: "510300"
      name: "华泰柏瑞沪深300ETF"
      index: "沪深300"
      category: "宽基"
    # 添加新ETF...
    - code: "159745"
      name: "国泰中证新能源汽车ETF"
      index: "新能源车"
      category: "行业"
```

添加后重新运行数据初始化：

```bash
python scripts/init_data.py
```

## 功能模块说明

### 市场概览（首页）
- 显示主要指数的 PE 百分位、股息率
- 市场温度计（综合估值热度）
- 一键判断当前是"低估"还是"高估"

### 估值看板
- PE/PB 百分位追踪（基于真实历史数据）
- 股债性价比（FED 模型）：ERP = 盈利收益率 - 国债利率
- 指数 PE 历史走势图

### 定投管理
- 支持三种定投策略：普通定投、估值定投、均线偏离定投
- 定投信号实时生成
- 新建/管理定投计划

### 网格交易
- 网格参数设计器（等差/等比/ATR动态）
- 可视化网格线分布
- 网格回测分析

### 轮动信号
- 大小盘轮动（根据 `strategies.yaml` 配置的标的）
- 风格轮动、红利轮动
- 股债轮动（基于 ERP）
- 行业动量轮动

### 组合管理
- 从 `portfolio.yaml` 自动读取持仓配置
- 实时显示各 ETF 最新价格
- 再平衡计算：输入当前市值，自动计算偏离和调仓建议
- 近60日收益率对比图
- 风险监控（回撤预警、仓位限制）

### 策略回测
- 选择 ETF + 策略类型 + 参数
- 回测引擎模拟历史交易
- 输出：收益曲线、夏普比率、最大回撤、月度收益热力图

### 投资报告
- 自动生成周报/月报
- 包含估值快照、ETF表现、行业动态

### 智能资讯
- 行业新闻采集与情绪评分（需配置 DeepSeek API Key）
- 政策追踪与影响评估
- 基本面变化趋势

### 持仓设置
- 可视化管理 ETF 持仓配置（增删改 ETF、调整目标权重）
- 支持一键均分权重
- 编辑再平衡参数和风控阈值
- 修改后直接保存到 `config/portfolio.yaml`

## 数据更新

### 手动更新

```bash
python scripts/daily_update.py
```

### 自动调度

系统内置 APScheduler 调度器（`src/scheduler/runner.py`），可通过以下方式启动：

```bash
python -m src.scheduler.runner
```

或在 Dashboard 侧边栏使用"数据管理"面板手动触发刷新。

```bash
uvicorn src.api.main:app --reload
```

如需使用 LLM 新闻分析功能，设置环境变量：

set DEEPSEEK_API_KEY=your_api_key

## 常见问题

### Q: 如何添加新的 ETF？
修改 `config/etf_universe.yaml` 添加 ETF 信息，然后运行 `python scripts/init_data.py` 拉取数据。

### Q: 组合管理的实际持仓从哪里来？
目前系统不自动同步券商账户。在"组合管理 > 再平衡分析"页面手动输入各 ETF 的当前市值，系统会计算偏离并给出调仓建议。

### Q: 数据源是什么？
主要使用 AkShare（免费开源），数据来源包括：
- ETF 日线：新浪财经
- 指数估值：中证指数公司
- 国债收益率：中国人民银行 / 美联储

### Q: 系统支持实盘交易吗？
不支持。本系统仅用于研究、回测和信号提示，不具备自动下单功能。

### Q: 如何修改定投金额和频率？
编辑 `config/strategies.yaml` 中的定投策略参数。
